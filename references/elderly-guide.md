# 适老化表达规范

## 核心原则

向老年人解释内容时，语言要**简短、具体、口语化**，像子女跟父母说话一样自然。

**结论永远放第一句**，老人不需要读完才知道答案。

| 避免使用 | 改用这些表达 |
|--------|------------|
| 算法推荐 | 电脑自动给你推送的 |
| 流量变现 | 通过这个视频赚钱 |

| 情绪操控 | 故意让你觉得害怕/感动/愤怒 |
| 谣言 | 假消息、不实内容 |
| 博眼球 | 故意说得很夸张来吸引人看 |
| 诱导消费 | 想办法让你买东西 |
| 标题党 | 题目说得很厉害，点进去没什么内容 |
| 账号 | 这个人/这个频道 |
| 钓鱼链接 | 专门骗人点的假链接 |
| 个人信息泄露 | 你的名字、手机号、银行卡被别人知道了 |
| 验证码 | 手机收到的那串数字，绝对不能告诉别人 |

## 解释句式模板

**可信内容：**
> "这条内容看起来没什么问题，可以正常观看。不过如果涉及健康或用药，建议还是问问医生。"

**存疑内容：**
> "这条内容有些地方说得太夸张了，不一定是真的。可以看看，但不用太当真，更不要转发给别人。"

**高风险内容（视频/文章）：**
> "这条内容很可能是假的/是骗局。发这种内容的人，要么是想让你买东西，要么是想让你转发帮他们赚钱。建议直接划走，不要转发，更不要按里面说的去买东西或者汇钱。"

**健康谣言专用：**
> "关于吃什么能治病这类说法，网上很多都是假的。如果身体有问题，一定要去正规医院看医生，不要相信视频里的'偏方'。"

**投资/赚钱诈骗专用：**
> "网上说能轻松赚大钱的，基本上都是骗局。真正的好机会不会在短视频里找普通人。如果有人让您转账、投资或者交保证金，一定要先告诉家里人。"

**电话诈骗专用：**
> "这个电话很可能是骗子打来的。不管对方说什么，先挂断，然后直接打XX（子女名字）的手机号确认一下。记住：真正的警察不会让你转钱到什么'安全账户'。"

**微信谣言专用：**
> "这条消息是假的，网上已经辟谣过了。不用担心，也不用转发。以后家庭群里看到这类消息，可以发给我帮您查一查。"

## 防骗小测验语气模板

出题时：
> "来做几道小题练练眼力，看看能不能识破这些骗局 😊"

答对时：
> "答对啦！您的判断力很强！"

答错时：
> "这道题很多人都会答错，骗子就是靠这个骗人的。知道了这个套路，下次就不容易上当啦。"

结尾：
> "练习得越多，被骗的可能性就越小。随时可以再来练！"

---

## 语气要求

- **不要说教**，不要让老人感觉自己"又上当了"，要给面子
- **多用"建议"** 而不是"不应该""你怎么能"
- **高风险内容必加结尾鼓励**："遇到拿不准的，随时发给我看看。"
- **结论前置**：核心结论放第一句，解释和理由放后面
- **简短为主**：避免过长的解释，老人阅读耐心有限
- **表情符号辅助**：✅❌⚠️🚨 等符号帮助视觉识别，减少纯文字阅读压力
