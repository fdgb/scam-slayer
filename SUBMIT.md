# 如何向「营销号亲爹」提交样本

你手里有疑似营销号 / 谣言 / 诈骗 / AI 伪造的内容，想让它进知识库、帮助更多人识别？
**提交完全自愿，且我们会在入库前做脱敏处理，不会留存你的个人信息。**

## 三种提交方式（任选其一）

### 方式 A：直接贴给 Expert（最省事）
在 WorkBuddy 里对营销号亲爹说：
> 「我有个样本想提交：<贴内容/链接/截图文字>」

专家会帮你整理成标准格式，并提示你确认提交。
（注意：通过聊天提交不会自动入库，作者会周期性人工复核。）

### 方式 B：GitHub Issue（推荐，可追溯）
打开仓库 Issues → New Issue → 选「提交样本」模板，按字段填：
- 内容类型：视频/文章/微信/电话/链接/其他
- 原始内容（或摘要）
- 你怀疑的点
- 是否已核实为 scam：是/否/不确定

### 方式 C：腾讯问卷收集表（如果你没有 GitHub 账号）
仓库 README「如何贡献样本」里放有腾讯问卷收集表链接，手机点几下即可填，**无需注册、真正匿名**。
作者每周导出 CSV 并半自动搬进待审队列，经审核脱敏后入库。

## 隐私与脱敏说明

- 提交即表示你同意我们**在脱敏后**将样本的模式/特征纳入公开知识库。
- 我们会删除：姓名、电话、微信号、具体地址、可定位个人的截图。
- 我们**不会**收集：你的 WorkBuddy 账号、聊天记录、设备信息。
- 你随时可以撤回：在 Issue/表单中说明，或联系作者。

## 提交后会发生什么

1. 作者人工审核样本真实性；
2. 提取「模式/特征」并脱敏，写入 `data/` 对应文件；
3. 跑 `build_manifest.py` 生成新版本；
4. 推送后，**所有已安装实例下次启动会自动热更新**到新版本。
