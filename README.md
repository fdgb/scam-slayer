## 怎么使用？

### 📦 第一步：安装专家包（一次性）

本仓库**本身就是一个可安装的 skill 包**——点右上角绿色 `Code` → `Download ZIP` 就能拿到（里面 `SKILL.md` 就在根目录，不用再去找单独的 skill 文件）：

1. **下载**：点仓库右上角绿色 `Code` 按钮 → `Download ZIP`（或 `git clone` 本仓库）。
2. **安装（任选其一）**：
   - **A. 手动放置（最稳，推荐）**：解压得到 `scam-slayer-main/` 文件夹 → **重命名为 `scam-slayer`** → 整个文件夹移动到 `~/.workbuddy/skills/scam-slayer/`（最终确认路径 `~/.workbuddy/skills/scam-slayer/SKILL.md` 存在即可）→ 重启 WorkBuddy，专家列表即出现「营销号亲爹」。
   - **B. 用 expert-manager 技能**：在 WorkBuddy 安装 `expert-manager` 技能，把解压后的文件夹注册进本地市场即可。
   - **C. 可视化导入**：若你的 WorkBuddy 版本支持「专家 → 导入本地专家」直接选文件夹，选解压后的 `scam-slayer/` 即可；若只接受 `.zip`，把 `scam-slayer/` 压缩成 zip 再导入（务必保证 zip 内 `SKILL.md` 在根目录）。
3. 安装后即可使用。**知识库自动热更新**：安装后无需重装，skill 每次会话会自动拉取 GitHub 最新知识库（仅 skill 逻辑层升级需偶尔重装覆盖）

### 💬 第二步：开始对话（已安装后）

打开 WorkBuddy，选择「营销号亲爹」，把你想鉴别的任何内容贴给它：

- 视频链接（支持 B站/抖音/快手/YouTube/IG/FB）
- 微信群截图文字
- 电话/短信内容描述
- 可疑链接或二维码
- 图片（怀疑是 AI 生成的）

### 给爸妈用的快捷方式

对 Expert 说：**「用能给爸妈看的方式说」**

它会输出超简洁卡片格式：

```
🔴 这是骗局！

XX视频在卖假药，说的"神医"根本不存在。
千万别信！更别转账！

→ 正确做法：去正规医院看病
```

这张卡片可以直接**截图发到家族群**。

## 🤲 如何贡献样本？（一起让知识库更强）

你遇到的可疑内容，就是帮别人避坑的关键数据。**每一条有效样本都可能阻止一个家庭受骗。**

### 提交方式（任选其一）

| 方式 | 适合谁 | 操作 |
|------|--------|------|
| **📝 腾讯问卷收集表** | 不想注册 / 老人友好 | [打开问卷](https://wj.qq.com/s2/27351181/485a/)（无需登录，手机点几下即可填）|
| **💬 直接跟 Expert 说** | 已安装用户 | 把可疑内容贴给专家，它会**帮你鉴别 + 整理成标准格式**；整理好的文字你再**贴到 GitHub Issue 或问卷**提交即可（这两条才真正送到维护者手里）|
| **📋 GitHub Issue** | 有 GitHub 账号 | [点这里提交](../../issues/new?template=submit-sample.md)（有模板，按提示填）|

> **隐私承诺**：所有提交经审核脱敏后才会进入公开知识库。我们不会在知识库中保留任何个人身份信息。

### 我们如何保护你的隐私

- ✅ 提交前请自行移除姓名/电话/微信号/地址等个人信息
- ✅ 所有样本经人工审核后会**脱敏处理**再入库
- ✅ 公开的知识库只保留「模式与特征」，不含任何个人身份信息
- ✅ 你随时可以要求撤回已提交的内容

## 🔄 知识库会持续更新吗？

**会的。** 作者每周审核新骗局样本并入库；已安装的 skill 会在每次使用时通过 `update_kb.py` 自动拉取最新知识库到本地，**你无需手动重装**。

> 💡 你提交的样本经审核脱敏后进入下一轮知识库，所有已安装用户下次使用时自动获得。

> ⚠️ **边界**：自动更新只覆盖“知识 / 数据层”（`data/*.md`）。若 skill 的鉴别**逻辑**本身升级（极少发生），才需重新下载安装包覆盖一次。

## ⚠️ 为什么这个项目很重要

在中国，每年因网络诈骗造成的损失以**百亿人民币计**。而其中很大一部分受害者是**中老年人**——他们信任屏幕上的文字，相信"专家"的话，分不清真广告和假科普。

营销号的套路在不断进化：从简单的"震惊体"标题，到 AI 生成的逼真视频，再到精准针对老年人的情感操控话术。**传统的反诈宣传跟不上骗子的迭代速度。**

这就是为什么我们做了一个**会持续学习、持续更新**的 AI 鉴别器。

它不是完美的。但它在努力追赶骗子的脚步。

而每一个提交样本的你，都在帮它跑得更快一点。
